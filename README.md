# a.chs.mod
一个“未知伤亡（Casualties Unknown）”的a.chs.mod多模组汉化MOD

目前：v0.0.3

直接把下载好的.dll文件放入游戏的"Casualties Unknown Demo\BepInEx\plugins"文件夹即可。
